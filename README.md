# Gomoku Game
A Python-based Gomoku (Five-in-a-Row) game with a Smart Machine opponent.

<img src="images/gomukuimage.png" alt="Gomoku Gameplay" width="600">

## How to Play
1. Install requirements: `pip install flask`
2. Run the app: `python gomukugameapp.py`
3. Visit `http://127.0.0.1:5000` in your browser.
